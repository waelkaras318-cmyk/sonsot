سياسة الخصوصية

نحن في منصة البطاقة الاجتماعية نلتزم بحماية خصوصيتك:

  • استخدام الكاميرا: يتم استخدام الكاميرا فقط لالتقاط صورة الهوية الخاصة بك وتخزينها محلياً للتحقق البيومتري.
  • البيانات: لا يتم مشاركة بياناتك الشخصية مع أي طرف ثالث دون استخدام الـ API Key الخاص بك.
  • الإعلانات: نستخدم ملفات تعريف الارتباط لعرض الإعلانات المناسبة لك عبر جوجل أدسنس.
